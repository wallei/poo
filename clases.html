Clases y Objetos en Python

¿Qué es una Clase?

En Python, una clase es una plantilla para crear objetos. Define las propiedades y los métodos comunes a todos los objetos de cierto tipo.

¿Qué es un Objeto?

Un objeto es una instancia de una clase. Es una entidad que tiene atributos y métodos específicos definidos por su clase.

Instanciación de un Objeto en Python

Para instanciar un objeto en Python, primero debes definir una clase utilizando la palabra clave class. Luego, puedes crear una instancia de esa clase utilizando la sintaxis nombre_objeto = NombreClase().

A continuación, un ejemplo de cómo definir una clase llamada Persona con un atributo nombre y un método saludar(), y cómo instanciar un objeto de esta clase:

class Persona:
    def __init__(self, nombre):
        self.nombre = nombre
    
    def saludar(self):
        print("Hola, soy", self.nombre)

# Instanciación de un objeto
juan = Persona("Juan")
juan.saludar()
